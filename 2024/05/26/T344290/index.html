环状序列

学校题单里头的一道题，2023年的6月第一次做这道题，往后隔一两个月回顾之前的题目时总会再次看看它，但始终没有做出来。近日想把之前的题单再刷一遍，尤其是把之前没有做出来的题再做一遍。很开心能看到自己做之前很多不会的题都更轻松了。也在昨晚想出了这道题的解法，今天写完并在此记录。

题目描述

给定长度 \(n\) 的环状整数序列 \(a_1, a_2, \ldots, a_n\)。对于任意元素 \(a_i\)（\(1 \le i < n\)），它的下一项元素是 \(a_{i+1}\)；特殊地，\(a_n\) 的下一项是 \(a_1\)。

现在，我们要将环状序列切割为一个有头有尾的长度 \(n\) 的一般序列 —— 首先选定 \(a_k\) 为序列第一个元素，然后依次将 \(a_{k+1}, a_{k+2}, \ldots, a_n\) 放在序列的末尾，之后再将依次 \(a_1, a_2, \ldots, a_{k-1}\) 放在序列的末尾。显然，选取不同的元素作为序列的第一个元素，我们将得到不同的长度 \(n\) 的序列。

请计算，在这 \(n\) 个序列中，有哪些序列所对应的前缀和序列是非负的。

输入格式

第一行包含整数 \(n\)，表示初始的环状序列的长度。

第二行包含 \(n\) 个整数 \(a_1, a_2, \ldots, a_n\)，描述环状序列。

输出格式

一个整数，表示满足要求的非环状序列的数量。

样例输入 #1

样例输出 #1

提示

【样例解释】

共有 \(4\) 个非环状序列的方案： - \(-3, 5, 1, 2\)（前缀和序列为 \(-3, 2, 3, 5\)） - \(5, 1, 2, -3\)（前缀和序列为 \(5, 6, 8, 5\)） - \(1, 2, -3, 5\)（前缀和序列为 \(1, 3, 0, 5\)） - \(2, -3, 5, 1\)（前缀和序列为 \(2, -1, 4, 5\)）

其中第 \(2 \sim 3\) 个满足要求。

【数据范围】

对于 \(25\%\) 的数据，\(n \le 10^3\)；
对于 \(75\%\) 的数据，\(n \le 10^4\)；
对于 \(100\%\) 的数据，\(1 \le n \le 10^6\)，\(|a_i| \le 10^3\)。

题解

由于数据范围是 \(10^6\) 这个级别，我们需要用 \(O(n)\) 或者 \(O(nlogn)\) 的算法来实现这道题。暴力的方法时间复杂度是 \(O(n^2)\) ：模拟每个非环状序列，并用前缀和检查是否符合要求。这种模拟的思路并不好优化，因为对于每一种非环状序列都需要对整个序列进行判定，这种判定已经是 \(10^6\) 的时间复杂度了，所以不好继续优化了。

如果一个问题无法正向地高效解决，也许我们可以换个角度想想。我原始的思路是这样的：如果这个序列全是非负数，那么这个序列就没有任何判定的必要，因为每一种非环状序列都符合要求。然而，如果有负数，那么它会影响它相邻的元素。因此，也许我们可以对序列中的每一个负数进行操作。

顺着这个思路，我们继续想。对于一个负数自己，我们可以确定的是以它为开头的序列一定是一个“非法”序列，那么这个负数是如何进而影响其他相邻的元素的呢？这个负数对相邻的元素的影响便只有两种情况：向左边的元素进行影响和向右边的元素进行影响。这里要注意，题目中这个序列的方向是一定的，即只能将序列的前面的部分移动到后面，不能出现反转这种操作。

向左边的元素进行影响时，每触及到某一个元素，都计算一个累加的和。如果累加和小于 \(0\)，说明以这些数为前缀序列的序列都是“非法的”。然而，究竟有几个序列能以这些数为前缀呢？显然这个序列是唯一的。换种说法，对于这个问题而言，一个序列的开头的第一个元素已经决定了这个序列。因此，只要一个负数能影响某些数，使得这些数的累加和小于 \(0\)，那么以这些数为前缀的序列，都是“非法序列”。如：\({a_i}\) 是一个负数，它所能影响到 \({a_j}, j \le i\)，那么有 \(i - j + 1\) 个序列是非法的，即以 \(j\) 到 \(i\) 之间（包括 \(j\) 和 \(i\) ）任意一个元素开头的序列都是“非法序列”。不过需要注意的是，这个区间中每一个元素都必须是内被 \(a_i\) 所影响的。否则，如果所累加的和大于 \(0\)，说明这个负数向左的影响力已经到头了。而也许某一些更左边的数能使前缀和为负，不难证明从 \(j\) 到那个数之间，一定存在负数。而那个负数就是独立于当前这个负数的，因此它需要被作为另一个负数单独考虑。

而向右边的元素进行影响，是没有意义的，因为所被影响的作为某一个序列的前缀的序列，所对应的那个序列是同一个序列，就像模拟那样。而这不是“找开头”这种思路所要求的。毕竟对于一个负数而言，以它开头的序列一定是一个“非法序列”。

当然这里还有包含的情况，比如一个负数向左进行影响时，遇到了另一个负数。能遇到这个负数说明了以下两点：第一、在遇到这个负数之前，正在考虑的这个原负数的影响从未中断。第二、这个负数不会使影响中断。也就是说，如果一个负数被另一个负数包含了，那么这个负数就不需要被单独考虑了。

对于环状问题，先倍长序列。

然后从后之前寻找负数并依次考虑。
